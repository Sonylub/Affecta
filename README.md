# Макеты интерфейса Affecta

Эта папка содержит макеты основных страниц приложения Affecta для демонстрации структуры и визуального представления интерфейса.

## Содержимое

- **entry_mockup.html** - Макет страницы дневной записи
- **dashboard_mockup.html** - Макет дашборда с графиками и статистикой
- **analytics_mockup.html** - Макет страницы аналитики
- **report_settings_mockup.html** - Макет страницы настройки отчёта

## Особенности

Все макеты созданы с использованием Tailwind CSS через CDN для быстрого просмотра. Макеты демонстрируют:

- Структуру страниц
- Расположение элементов интерфейса
- Визуальное оформление
- Основные компоненты системы

Макеты не содержат функциональности и служат только для визуального представления интерфейса.
